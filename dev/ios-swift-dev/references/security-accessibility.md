# 安全与可访问性

## 安全

### 敏感数据存储

规则只有一条：

- Token、密码、密钥、敏感凭据进 `Keychain`。

不要使用：

- `UserDefaults`
- 明文文件
- 可恢复的普通本地缓存

### 加密

- 新代码优先使用 `CryptoKit`。
- 对称加密、哈希、签名都应走现代 API。
- 不要手写不必要的密码学流程。

### 传输安全

- 默认遵守 ATS。
- 例外配置必须明确记录原因。
- 证书固定只在确有安全需求且维护能力足够时使用。

## 可访问性

### 基础要求

- 每个交互控件都有可读标签。
- 图标按钮必须有可访问名称。
- 纯装饰元素对辅助功能隐藏。
- 触控区域足够大。

### 动态字体与系统偏好

必须考虑：

- Dynamic Type
- Reduce Motion
- Increase Contrast
- Bold Text

### 焦点管理

- Sheet、弹窗关闭后，焦点应回到触发控件。
- 自定义模态需要明确焦点与退出行为。

## 快速判断

以下问题任意命中，都说明这部分还没做完：

- 关闭弹窗后，VoiceOver 焦点丢失。
- 图标按钮读不出作用。
- 依赖颜色区分状态，没有文字或图标辅助。
- 缩放字体后布局崩掉。

## 快速检查

- [ ] 敏感数据只进 `Keychain`。
- [ ] 加密方案使用 `CryptoKit` 或等效安全实现。
- [ ] 页面支持基础辅助功能阅读与操作。
- [ ] 动态字体和减少动态效果已考虑。
